Python API
=============

BFEE2 Python API provides information for the end-user to use BFEE in a console program / streamline.

One may always use **BFEE2.inputGenerator** and **BFEE2.postTreatment** when using BFEE2 Python API in automated/streamlined binding free-energy calculation.
If one wants to use BFEE as an NAMD, GROMACS, Colvars, or FEP parser, **BFEE2.templates_namd**, **BFEE2.templates_gromacs**, and **BFEE2.third_party** also deserve a look.
The GUI and AI-assistant support modules are documented for completeness.

See BFEE2 user guide if one wants to know how to use BFEE2.
